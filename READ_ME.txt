Small car racing game with AI, developed using Blender Game Engine - tested on Windows 8.1 with Blender 2.74

Follow these steps to try it:

1. Download and install Blender
https://www.blender.org/download/

2. Copy/paste "car_game_lib" folder into Blender Python folder:
C:\Program Files\Blender Foundation\Blender\2.74\python\lib\site-packages\

3. launch AICarRacingTest.blend file

4. In Blender:
By default, you should be in the Game Logic view.
Press P to start playing, ESC to stop the game.

5. Keys:
Control player car: keyboard W-S-A-D
Control Camera height/distance: Numpad 8-4-6-2
Switch Camera to next/previous car: Keyboard O-P
